ScholarStream is a modern web application designed to help students explore, apply for, and manage scholarships worldwide. The platform provides a seamless experience for discovering scholarships, submitting applications, making secure payments, and tracking funding opportunities, including full-funded and partial scholarships.

Features

Browse Scholarships
View a comprehensive list of global scholarships with detailed descriptions, eligibility criteria, and deadlines.

Apply for Scholarships
Students can easily submit scholarship applications directly through the platform.

Secure Payments
Pay application or processing fees safely via integrated payment gateways.

Track Funding Status
Monitor the status of scholarship applications and funding outcomes.

Full-Funded & Partial Scholarships
Filter and discover both fully funded and partially funded scholarship opportunities.

User Authentication
Secure registration and login system with password encryption.

Profile Management
Users can upload profile photos, update personal information, and manage their applications.

Social Login Integration
Login using Google or other social accounts for convenience.

Tech Stack

Frontend

React 19

TailwindCSS + DaisyUI

Axios for API requests

React Hook Form for form management

React Router DOM for routing

Framer Motion & Swiper for animations and sliders

Backend

Node.js with Express.js

MongoDB for database

Bcrypt for password hashing

JWT (optional) for authentication

Stripe for payment processing

Firebase Admin for user management

Installation

Clone the repository

git clone 
https://github.com/Fahmida0010/ScholarStream-frontend-.git
https://github.com/Fahmida0010/scholarStream-backend.git

Backend Setup

cd scholarStream-backend
npm install
nodemon index.js

Create a .env file with  MongoDB connection string, Stripe keys, and Firebase credentials.

Frontend Setup

cd scholarStream_frontend
npm install
npm run dev

Add a .env.local file with API URLs and  image hosting API key (like imgbb).

Usage

Register a new account with your email, password, and profile photo.

Login using your credentials or social login.

Browse available scholarships and check eligibility.

Apply for scholarships directly through the platform.

Make payments if required for specific applications.

Track your scholarship applications and funding status.
